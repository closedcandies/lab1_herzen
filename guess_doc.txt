"""
Этот скрипт реализует функцию бинарного поиска, чтобы определить, сколько операций потребуется для нахождения заданного значения в указанном диапазоне.

Функции:

1. binary_search(floor, ceil, value):
   - Выполняет бинарный поиск для нахождения целевого значения в заданном диапазоне.
   - Аргументы:
     - floor (int): Нижняя граница диапазона поиска.
     - ceil (int): Верхняя граница диапазона поиска.
     - value (int): Значение, которое нужно найти в диапазоне.
   - Возвращает:
     - operations (int): Количество операций, необходимых для нахождения целевого значения.
   - Описание:
     - Функция постоянно вычисляет среднюю точку диапазона (floor, ceil), пока не найдет целевое значение.
     - В зависимости от того, меньше или больше средняя точка значения, она корректирует нижнюю или верхнюю границу соответственно.
     - Отслеживает количество операций, необходимых для нахождения значения.

2. main():
   - Основная функция для выполнения скрипта.
   - Описание:
     - Запрашивает у пользователя три входных значения: значение для поиска, нижнюю и верхнюю границы диапазона поиска.
     - Вызывает функцию binary_search() и выводит количество сравнений, необходимых для нахождения значения.

Использование:
- Запустите скрипт и введите целевое значение, нижнюю и верхнюю границы при появлении запроса.
- Скрипт вернет количество сравнений, необходимых для нахождения значения с использованием бинарного поиска.

Пример:
```
Число, которое необходимо угадать: 37
Нижняя граница диапазона: 0
Верхняя граница диапазона: 100
Число отгадано. Количество сравнений: 7
```

Заметки:
- Значения `floor` и `ceil` должны быть допустимыми границами, при этом `floor < ceil`.
- Значение `value` должно находиться между `floor` и `ceil`.

"""
